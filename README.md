# java-advance-test-util

This lib has some utilities used in java testing.

## Features

- Annotation assertions
- Reflection utils

## Developing commands

Clean:
```
make clean
```

Build:
```
make jar
```

Test:
```
make test
```

Coverage:
```
make coverage
```

Javadoc:
```
make doc
```

Run demo:
```
make run
```
